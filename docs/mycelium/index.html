Mycelium — Cross-platform App


User Manual

Mycelium is a cross-platform longterm learning application powered by Mycel.
It is open-source and acts as a complete client, implementing all main functionalities of Mycel into a single app. It is currently available on Windows, Linux, and Android, and will be available for macOS and iOS soon.

Mycel Project and Mycelium are in early development. Visuals shown here may not reflect the current state of the application.

Install

Extract the downloaded archive and run the .exe file inside.

Make the downloaded .AppImage executable and run it:

chmod +x mycelium-linux-vX.X.X.AppImage && ./mycelium-linux-vX.X.X.AppImage

Replace vX.X.X with the actual version number.

Enable installation from unknown sources in your Android settings, then open the downloaded .apk file and follow instructions.

macOS support is coming soon.
