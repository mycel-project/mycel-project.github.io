# Mycelium

## Known limitations (temporary)

As Mycelium is currently in a very early stage, there are many limitations that are planned to be addressed. Here are some you may encounter:

- The only available option is to import via URL
- Complex rendering of Markdown and HTML elements is not supported yet (will switch to CodeMirror soon to address this)
- Search functionality has a button but is not yet implemented
- Nodes cannot be moved or reparented for now
- Long articles can be laggy. The current workaround is to split the article by heading
- Toasts (the notification popups at the bottom) are unstable: sometimes missing, sometimes duplicating. They're not critical to use the app, just informational. If something feels off, a restart usually helps. This will be fixed soon.
Multiline content is not supported in clozes

Note that Mycel already supports features that are not yet implemented in Mycelium.
