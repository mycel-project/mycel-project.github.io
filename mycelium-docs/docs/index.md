# Mycelium

## Known limitations 

As Mycelium is in an early stage, there are limitations that are planned to be addressed. Note that Mycel already supports features that are not yet implemented in Mycelium.

Here are some you may encounter:
### Learning
- The only available option is to import via URL
- Multiline content is not supported in clozes
- On mobile, selecting markdown elements can be quite capricious
### Navigation
- NodeTree (left drawer) won't save its scroll position or opened node.
- Cursor position won't be saved across nodes
### Misc
- Nodes cannot be moved or reparented
- Toasts (the notification popups at the bottom) are unstable: sometimes missing, sometimes duplicating. They're not critical to use the app, just informational. If something feels off, a restart usually helps. This will be fixed soon.
