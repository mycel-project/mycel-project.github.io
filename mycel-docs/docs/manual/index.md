# Mycel Manual
This manual is intended for users, developers, and integrators looking to understand, use, extend, or implement Mycel in external tools.

To learn how to use Mycel in practice, refer to the [Mycelium manual](https://mycel-project.github.io/docs/mycelium/) or the documentation specific to your [implementation of Mycel](https://github.com/mycel-project/awesome-mycel).
