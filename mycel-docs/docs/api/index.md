A live version using Scalar is avalaible at [MycelCloud](https://api.mycelcloud.com/scalar).

This interface provides a significantly enhanced and more intuitive user experience. Should the cloud version be unavailable, you can always access it locally on any self-hosted Mycel instance by navigating to mycel-ip:port/scalar.

The API contract will be considered stable starting from Mycel 1.0.0. Before that, breaking changes may occur at any time.
